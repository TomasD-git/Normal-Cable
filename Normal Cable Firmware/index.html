Command Reference
STRING <text> - Type text
Example: STRING Hello World!
KEY <name> - Press key
Example: KEY ENTER | KEY TAB
DELAY <ms> - Wait milliseconds
Example: DELAY 1000 (default: 5000)
PRESS <mod>+<key> - Modifier combo
Example: PRESS CTRL+C | PRESS WINDOWS+R
REPEAT <n> - Repeat next command n times
Example: REPEAT 5 then STRING A (types AAAAA)
REM <text> - Comment
Example: REM This is a comment
